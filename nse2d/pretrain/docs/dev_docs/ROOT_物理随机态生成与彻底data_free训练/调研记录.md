## 冲突与缺口

- 证据冲突：
  - staged 路线从经验结果看最好，但它不是 strict data-free，本质上仍依赖真实状态锚点。
  - learnable GRF 与 self-training bootstrap 都能改善一些数值，但没有解决“multi-step 越滚越坏”。
- 关键缺口：
  - 缺一个比独立 GRF 更强、但又不直接依赖真实未来轨迹的随机态生成方案。
  - 缺 `batch_size` 的系统性对比。
  - 缺当前 rollout 爆炸 pattern 的结构化归档与分 field 分析。

## 当前结论

- 可直接采用：
  - 主问题应写成“更符合物理的随机态生成与彻底 data-free 训练”。
  - 当前候选方案可先分三类：
    - 生成模型
    - learnable GRF
    - self-training bootstrap
- 暂不能下结论：
  - 哪一条最终会成为主方案
  - `n` 与 `Ti` 场绝对值限制是否应彻底去掉
  - 爆炸 pattern 是否已经说明 learnable GRF 这条路本质上不行
