> 硬规则：这里只写当前活跃面。任何单条记录超过 8 到 12 行，默认下沉到对应 `Qxx/`。

## 目标

- 根问题：针对复杂 PDE 动力过程，能否尽量不依赖真实仿真数据，只靠物理约束与随机生成训练态，训练出可用且尽量长期稳定的 data-free surrogate model。
- 当前具体场景：2D 5-field MHD surrogate training。
- 验收指标：
  - 主指标：MHD 与 GRF 测试集在 step-1/3/5/10 的 per-channel mean relative L2
  - 现象指标：不再长期出现“train loss 与单步变好，但多步 rollout 变坏”
  - 协议指标：train/inference 路径一致、监督时间尺度对齐、val/test 角色拆分

## 研究的问题（主问题已明确；子问题待后续补充）

### 根问题：物理随机态生成与彻底 data-free 训练

- 已有的结论和观察：
  - 当前最好经验路线仍然是“GRF 预训练 + simulation data 继续训练”。
  - 当前 raw GRF / raw online 路线中，常见现象是：`train loss` 持续下降，单步 eval 下降，但多步 rollout 反而越来越差。
  - 当前最可信判断是：真实状态锚点仍然关键，而当前随机态仍不够物理。
- 当前重点不是“再加一个 trick”，而是判断什么样的随机训练态、训练协议和路线设计，才可能真正逼近 strict data-free。
- 当前子问题状态：先不在这里提前固化；后续按“调研 / 深度思考 / 实验验证 / 理论推导 / Agent讨论 / 老师或同事讨论”六类，与你逐步确认后补上。
- 入口：`ROOT_物理随机态生成与彻底data_free训练/README.md`
