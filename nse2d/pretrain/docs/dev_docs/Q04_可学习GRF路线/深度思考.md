# 深度思考

> 何时使用：有多条路线、关键假设未定、需要第一性原理分析或反例攻击时创建。

## 元信息

- QID：Q04
- 最近更新时间：2026-03-11
- 当前要回答的核心问题：
  - learnable GRF 是否真的优于 fixed prior，还是只是配合 `soft-Linf` 后看起来没那么差？
  - `exp20` 的收益到底来自 prior 变得可学习，还是来自局部稳定化手段？
  - 为什么 `exp21` 在中短期更好，却没能把长期 rollout 救回来？
  - Q04 应继续作为主线推进，还是降级为 bridge / baseline？

## 已确认事实

- 事实 1：`exp14 (learnable GRF)` 是高价值失败，MHD / GRF Mean 达到 `301.3 / 346.0`。
- 事实 2：`exp20 (learn-GRF + soft-Linf)` 相比 `exp14` 有数量级改善，但 MHD / GRF Mean 仍有 `22.21 / 22.67`，远高于 anchored baseline。
- 事实 3：`exp21 (learn-GRF + soft-Linf + multi-step)` 的 `step-3` / `step-5` 优于 `exp20`，但 `step-10` 与 `Mean` 更差，因此当前证据不支持“multi-step 救回长期 rollout”。
- 事实 4：`exp_offline_pde (1GPU)` 与 `exp1b (1GPU)` 都在 `0.4 / 1.0` 左右的可用区间，说明 learnable GRF 当前离“可用物理流形”还很远。
- 事实 5：当前 learnable GRF 仍是 low-dimensional per-field 参数化，缺少五场联合结构；这与 Q04 想回答的“能否把 `q` 推近 `\mu`”之间还存在明显能力缺口。

## 核心假设

- 假设 A：learnable GRF 可以修正 fixed GRF 的部分尺度或谱形错配，但当前参数化只能做低阶、per-field 的局部修补，不足以恢复五场联合结构。
- 假设 B：`exp20` 的收益并不等于“learnable prior 已经成立”；其中相当一部分改善可能来自 `soft-Linf` 对训练稳定性的帮助。
- 假设 C：在支持集仍然错误的前提下加入 multi-step PDE，可能会改善中短期 rollout，却把优化重点推向对远期 rollout 更不利的区域。
- 假设 D：如果 matched fixed-vs-learnable 对照仍看不到 `step-10` / `Mean` 的明确收益，那么 Q04 更适合长期保留为 bridge / baseline，而不是主线。

## 反例攻击

- 最脆弱的前提：
  - “只要把 GRF 做成可学习，训练分布就会自然向真实 rollout 分布靠近。”
- 什么现象一旦出现就说明这条路很难升格为主线：
  - 在与 fixed prior 完全 matched 的设置下，learnable prior 仍无法稳定优于 fixed prior 的 `step-10` / `Mean`。
  - 即使继续增强 prior 参数化，结果仍明显落后于 anchored baseline 的近邻参照。
  - 后续实验持续复现“`step-1` 或 `step-3` 改善，但 `step-10` 不改善甚至更差”。
- 目前最强反对意见：
  - 当前看到的收益主要来自 `soft-Linf` 或训练协议，而不是 learnable prior 真正在改善 `q -> \mu`。

## 路线比较

| 路线 | 为什么可能成立 | 最大风险 | 当前判断 |
|------|----------------|----------|----------|
| raw learnable GRF（`exp14`） | 最直接检验“可学习先验”是否有价值 | 没有稳定化手段时直接灾难性失稳 | 已是高价值失败，不建议继续原样投入 |
| learnable GRF + `soft-Linf`（`exp20`） | 可能把先验学习与局部稳定化结合，至少把路线拉回可分析区间 | 改善可能主要来自 `soft-Linf`，而不是 prior 真变好了 | 适合作为 bridge / baseline，值得保留 |
| learnable GRF + `soft-Linf` + multi-step（`exp21`） | 试图把训练目标更靠近 rollout 过程 | 当前只改善中程，不改善远程；容易误判为“更物理” | 现有证据偏负，不宜继续堆同类 trick |
| 更强 joint learnable prior（待设计） | 更直接攻击五场联合结构不足这个瓶颈 | 实现成本更高，且若无 matched 对照仍难解释收益 | 若 Q04 还要继续，优先级应高于再调 loss 权重 |

## 当前判断

- 当前更倾向的路线：
  - Q04 仍值得保留，因为 `exp20` 说明“可学习先验”不是完全无效。
  - 但它当前更像 bridge / baseline，而不是 pure online 主线。
  - 如果继续投入，优先级应是 matched 对照和更强 prior，而不是继续堆更多 loss trick。
- 暂不推进的路线：
  - 不再仅围绕 `soft-Linf` 权重、multi-step 组合做大量试参。
  - 不把 `exp20` 的局部改善直接上升为“Q04 已证明成立”。
