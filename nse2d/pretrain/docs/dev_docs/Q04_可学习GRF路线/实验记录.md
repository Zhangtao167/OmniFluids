## Benchmark 与配置

- 当前 benchmark：
  - MHD test 与 GRF test 双指标并行。
  - 常用汇报步：`step-1 / 3 / 5 / 10`，`Mean` 为 4 个关键步平均。
- 关键配置：
  - Q04 的所有“变好”都必须同时报告 MHD / GRF 与 `step-10` / `Mean`，不能只看 `step-1`。
  - 当前核心配对是 `exp14 -> exp20 -> exp21`。
- 对照组或 baseline：
  - `exp14 (learnable GRF)`
  - `exp_offline_pde (PDE loss, 1GPU)`：MHD / GRF Mean `0.4097 / 1.060`
  - `exp1b (sup dt=1s, 1GPU)`：MHD / GRF Mean `0.4099 / 1.030`
  - 当前缺少 matched fixed-vs-learnable 对照，这是 Q04 最重要的证据缺口

## 关键结果

- 已确认结果：
  - `exp20` 相对 `exp14` 是数量级改善：MHD Mean `301.3 -> 22.21`，GRF Mean `346.0 -> 22.67`。
  - `exp21` 相对 `exp20` 的 MHD `step-3 / step-5` 从 `9.436 / 19.92` 改到 `2.424 / 14.71`，GRF `step-3 / step-5` 从 `11.25 / 22.70` 改到 `4.736 / 21.50`；但 MHD / GRF `step-10` 从 `58.94 / 56.27` 恶化到 `85.21 / 83.29`，Mean 也从 `22.21 / 22.67` 恶化到 `25.72 / 27.49`。
  - 即使是当前更好的 `exp20`，也仍远差于 anchored baseline，说明 learnable GRF 还没有把训练分布推到可用物理流形附近。
- 暂不确定结果：
  - `exp20` 的主要收益来自 learnable prior 还是 `soft-Linf`，当前无干净 ablation。
  - `exp20` 与 `exp21` 都已停止，因此当前只能基于现有曲线与评估判断阶段性结论，不能再期待后续训练自动扭转。
  - 当前参数化是否已到上限，还是更强 joint prior 仍有空间。
- 高价值失败：
  - `exp14` 证明了“可学习”本身不是充分条件。
  - `exp21` 证明了“加 multi-step”也不是充分条件。

## 结果影响评估

- 这轮结果改变了什么判断：
  - Q04 不应被直接判死；`exp20` 至少说明 learnable prior 可以把路线从灾难性失败拉回可分析区间。
  - 但 Q04 也不应升格为当前主线；现阶段更合适的定位是 bridge / baseline。
- 不改变什么判断：
  - 真实状态锚点仍然关键。
  - 当前 one-step 或中短期改善不等于长期 rollout 已经被解决。
